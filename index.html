実務で使える6つのレシピ

セッション開始通知 / 危険コマンドブロック / 自動フォーマット / ログ記録 / タスク完了通知

このページについて
Claude Code Hooks（ツール実行の前後に任意のシェルコマンドを自動実行できる機能）を実務で使える形にまとめました。 コピー&ペーストですぐ使えるスクリプト付きです。 ソースコード: GitHub リポジトリ

🪝 実装したHookの一覧

SessionStart

セッション開始サマリー

Claude Codeを起動したとき、プロジェクト状態を自動表示

  • 現在のGitブランチ表示
  • 未コミット変更件数
  • 最新コミットメッセージ
  • .envファイルの存在チェック
PreToolUse

保護ファイルガード

編集前に機密ファイルへのアクセスをブロック

  • .env / .env.production をブロック
  • secrets.json / credentials.json をブロック
  • package.json 変更時に警告
  • ブロック理由をClaude Codeに通知
PreToolUse

危険コマンドブロック

Bash実行前に破壊的コマンドをインターセプト

  • rm -rf / 系コマンドをブロック
  • dd if=/dev/zero 等をブロック
  • 本番デプロイコマンドで確認
  • パターンマッチで柔軟に拡張可能
PostToolUse

自動フォーマット

ファイル編集後に対応フォーマッターを自動実行

  • JS/TS/JSON/CSS → Prettier
  • Python → Black または Ruff
  • Shell → shfmt
  • フォーマッターが無い場合はスキップ
PostToolUse

Bashコマンドロガー

実行したコマンドをすべてログファイルに記録

  • ~/.claude/bash-history.log に記録
  • タイムスタンプ + 終了コード付き
  • 200文字で切り詰め（ログが肥大化しない）
  • あとからClaude Codeの操作を追跡可能
Stop

タスク完了通知

Claude Codeがタスクを完了したらデスクトップ通知

  • macOS: osascript で通知
  • Linux: notify-send で通知
  • ターミナルベルでフォールバック
  • エラー停止時は別メッセージ

⚙️ settings.json の設定

.claude/settings.json 
{
  "hooks": {
    "SessionStart": [{
      "hooks": [{
        "type": "command",
        "command": "bash scripts/hooks/session-start.sh"
      }]
    }],
    "PreToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [{ "type": "command", "command": "bash scripts/hooks/pre-edit-check.sh" }]
      },
      {
        "matcher": "Bash",
        "hooks": [{ "type": "command", "command": "bash scripts/hooks/pre-bash-guard.sh" }]
      }
    ],
    "PostToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [{ "type": "command", "command": "bash scripts/hooks/post-edit-format.sh" }]
      },
      {
        "matcher": "Bash",
        "hooks": [{ "type": "command", "command": "bash scripts/hooks/post-bash-logger.sh" }]
      }
    ],
    "Stop": [{
      "hooks": [{ "type": "command", "command": "bash scripts/hooks/stop-notify.sh" }]
    }]
  }
}
